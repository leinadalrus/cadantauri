## Read-me

Cadantauri is a personal project employing data-oriented design - while still being agnostic to object-oriented and functional programming.

The whole idea is to make developing an audio editor fun, and so I had chosen to employ Zig (0.15.2 as of writing this,) in order to make the developer experience fun from a low-level up to the high-level.

This way if you wanna employ C or C++ for additional performance, or just want to do foreign-function-interfacing or interoptability then you can with Zig's means of generating a C or C++ compiler.
